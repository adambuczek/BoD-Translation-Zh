# 安装 Instalation

1. 下载 https://github.com/xuchkang171/BoD-Translation-Zh.git
2. 将压缩包内 BoD-Translation-Zh-master 文件夹中的 lang_zh 和 bod_lang_zh_demo 两个文件夹复制；
3. 打开 Steam 的「库」。
4. 右键点击 Book of Demons，选择「属性」->「本地文件」->「浏览本地文件」。
5. 将复制的两个文件夹粘贴到这里。
6. 重启游戏，在游戏的设置中切换为简体中文即可。
